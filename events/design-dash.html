DESIGN DASH

Unleash your creativity to build stunning UI designs from scratch.

Event Description

Design Dash is a UI/UX design challenge where teams create innovative user interfaces based on a provided problem statement. Showcase your creativity and design thinking!

Team Size

2 - 3 Members

Objective

Create a complete UI design based on a given problem statement using any modern design tool.

Rules & Regulations

  • Allowed design tools: Figma, Adobe XD, Canva, etc.
  • No use of pre-built templates is allowed.
  • Participants must justify their design choices to the jury.
  • Judging will be based on creativity, usability, and aesthetics.
