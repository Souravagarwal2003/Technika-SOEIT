SUDOKU

Challenge your mind with a classic number puzzle.

Theme

Problem Solving

Event Description

It is a single player event conducted on PC. Instructions will be provided on the event day. The game involves solving a 9x9 Sudoku puzzle within a specified time limit. Points are awarded based on accuracy and speed.

Guidelines

  • Maximum Team Size: Individual
  • Time-limited Sudoku game on PC.
  • Unfinished games will be auto-submitted and locked.
  • 3 hints allowed per player with 5 point deduction per hint.
  • Each correct box value gives 5 points.
  • If multiple players have same points, completion time will decide the winner.

Judgement Criteria

  • Points scored by the contestant.
  • In case of a tie, completion time will be the deciding factor.

Disqualification Criteria

  • Using any online Sudoku solver tools.
  • Discussing with other participants during the game.
