CODE BUSTERS

Unleash your coding skills in an exciting challenge.

Theme

Coding & Problem Solving.

Team Details

Maximum Team Size: 2 members (Solo or Duo)

Guidelines / Event Rules

  • This is a three-round, two-day, solo/group (max 2 people) frontend coding adventure using HTML and CSS.
  • In each round, players will be eliminated based on score.
  • Only the winners advance to the final round.
  • No use of mobile phones or internet in the first two rounds.

Rounds

  1. Quiz Round – Time: 15 min
    30 MCQs, 1 mark each. Minimum 12 marks required to qualify.
  2. Front-End Design Round – Time: 30 min
    Code a sample web page using HTML/CSS. Match must be at least 70%.
  3. Final Problem Solving Round – Time: 3 hours
    Solve a real-world college problem using a demo website/app (e.g., canteen item availability checker).

Judgement Criteria

  • Team scoring the highest collective points across all rounds wins.

Disqualification Criteria

  • Violation of discipline during competition leads to direct elimination.
